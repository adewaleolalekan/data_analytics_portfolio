Cleaning Data with
Sorting & Filtering

Preparing a student performance dataset for analysis using sorting and filtering techniques in spreadsheets.

Invalid data removed
Missing values handled
Data transformed for analysis

Project Scenario

As a data analyst for a large public school district in Portugal, I was tasked with preparing a dataset for an upcoming analysis. The superintendent wanted to understand the factors affecting student grades in two high schools: Gabriel Pereira (GP) and Mouzinho da Silveira (MS).

The Challenge

The raw dataset — collected from academic reports and student surveys — contained student grades, background information, and study habits. Before any analysis could be performed, it was critical to clean the data. An analysis built on dirty data could lead to flawed conclusions and ineffective policy changes.

Data Cleaning Process

01
Initial Exploration with Sorting

Performed a multi-level sort on the dataset: first by school (A–Z), then by age (Z–A). This immediately revealed a problem — the age range showed students as old as 22, which was suspicious for a high school dataset and likely represented data entry errors.

02
Removing Invalid Data with Filtering

After confirming with the superintendent that the valid age range was 15–19, I applied a filter to the age column to isolate entries with ages 20, 21, and 22. The nine affected rows were deleted, ensuring the age data was accurate.

03
Handling Missing Values

The reason column (why a student chose their school) was important for analysis but contained blank cells. I filtered for blanks and filled each one with a consistent placeholder — none_given — using the fill handle to apply the change efficiently across all affected rows.

04
Transforming Data for Analysis

The Medu (mother's education) and Fedu (father's education) columns were stored as text (e.g. "higher education"), which is unsuitable for statistical calculations. I used Find & Replace to systematically convert each education level to a numeric code (e.g. "higher education" → 4) across both columns.

Project Outcome

Through this methodical process of sorting, filtering, and transformation, the student performance dataset was cleaned into a reliable, accurate, and properly formatted state — ready for the analysis phase.

Key takeaway: Foundational data cleaning work like this is what ensures that insights derived for the school district are valid and can lead to effective, data-driven decisions. Garbage in, garbage out — clean data is the starting point for everything.
